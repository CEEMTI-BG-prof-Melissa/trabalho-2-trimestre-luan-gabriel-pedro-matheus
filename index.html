O que é POO?

POO é um modelo de programação que organiza o código com base em objetos

Princípios Fundamentais

  • Encapsulamento: Protege os dados e permite acesso controlado por métodos.
  • Herança: Permite que uma classe herde características de outra.
  • Polimorfismo: Permite que objetos se comportem de maneiras diferentes dependendo do contexto.
  • Abstração: Foca apenas nos detalhes essenciais de um objeto.

Vantagens da POO

A POO melhora a organização do código, facilita a manutenção, promove o reuso e torna os sistemas mais robustos e escaláveis.
